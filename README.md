# Pupperazzi

Full-stack pet social platform that connects dog owners through profiles, messaging, and local events.

## Tech Stack
- React (Vite)
- PHP
- MySQL
- REST APIs

## Features
- User authentication and profiles
- Dog profile creation and matching
- Messaging system between users
- Event creation and RSVP functionality

## Overview
Pupperazzi is designed to help dog owners connect through shared interests, enabling social interaction and event participation in a clean, responsive interface.
